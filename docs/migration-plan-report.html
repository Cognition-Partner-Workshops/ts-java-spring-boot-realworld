1. Executive Summary

This migration plan outlines the upgrade of the spring-boot-realworld-example-app from Java 11 to Java 21, converting from a blocking servlet-based architecture to a fully reactive stack using Spring WebFlux.

Primary KPIs

  • Java version: 21
  • Reactive framework: Spring WebFlux / Reactor
  • Code coverage: ≥95% (line + branch)
  • Sonar: Quality Gate PASS
  • 42Crunch: API Security Gate PASS
  • No breaking functionality: 0 critical regressions
